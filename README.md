# yimijianfang
MyScroll，一款在顶部显示浏览进度同时支持丰富滚动事件的原生js插件。

## [演示地址](https://yimijianfang.github.io/MyScroll)

### 用法：
1. 引入MyScroll.js

2.
```
new MyScroll({
  "showBar": true, //是否显示进度条
  "color": "rgb(10, 116, 218)", //进度条颜色
  "height": "3px", //进度条高度
  "debug": false, //调试模式
  "event": []
});
```

### 可以实现：
.
1. 顶部显示浏览进度

2. 滚动到底部刷新数据

3. 埋点动画

4. ...
